# 🧼 Mopen's Liquid Soap Website

Welcome to the official website project for **Mopen's Liquid Soap**, a proudly Cameroonian business based in **Bafoussam, Cameroon**. This website was built to showcase our products, provide customers with an easy way to place orders, and present our brand professionally online.

---

## 📖 About the Project

Mopen's Liquid Soap is dedicated to providing high-quality, affordable, and effective liquid soap for homes, restaurants, hotels, schools, offices, and businesses.

This website allows customers to:

- View available liquid soap products
- Check product prices
- Learn about the company
- Read customer testimonials
- Find answers to frequently asked questions
- Order via MTN Mobile Money, Orange Money, and WhatsApp
- Contact the business easily

---
